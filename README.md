# colorizr
## Description
colorizr is a ruby gem which colors the terminal output.
##Â Installation
```
$ gem install colorizr
```
## Usage
### Basics
To use colorizr:
```
require 'colorizr'
```
To color the output use `"mystring".color_name`:
```
"This is red".red
"This is blue".blue
```
Class methods:
```
String.colors
String.sample_colors
```
